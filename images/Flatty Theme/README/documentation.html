» Getting Started


1.) Before you can actually start using and configure Flatty you have to upload all the files inside the Upload folder.




2.) Now head to your Admin Control Panel  »  Templates & Styles  » and click on Import a Theme



2.1) Afterwards click on the "Choose File" button and search for the Flatty-theme.xml file which is located right next to the Upload folder (as seen on the screenshot above).

Give the theme you've just imported a name of your choice, check "Ignore Version Compatibility" and then click on "Import Theme".
