Flatty Documentation & Installation Guide
In order to install and setup the Flatty Theme properly you should defenitely take a look into this guide and follow all mentioned steps and instructions as good as possible. If you know what you're doing and what you need to do then you can skip this guide of course. If you have problems nevertheless then dont hesitate to ask for help via private message, or creating a threadat the MyBB support forums for further help. In case you like this theme, feel free to recommend it. Thanks.

2) I also recommend you and your users to use the Classic Post-Mode, since I haven't really optimized the Default Post-Mode  âˆ´  User CP  »  Edit Options  »  and check Display post in Classic Mode.

3) If you are using the Classic Postbit-Mode as default, I'm recommending you to use a default avatar size of minimum 100 x 100. The maximum should be around 120 x 120.


Small extra note


None of the current available plugins are actually supported by default besides from DVZ Shoutbox. Means that some are looking & working better and some does not. Therefore I'm not able to garantuee you that the plugin you are using is working (flawless) on your board.If it does not work flawless and some weird (positioning) bugs appear then please try reactivating the plugin(s) at first, if that didn't help save your edits you've done and reinstall the plugin(s) and if that didn't work either then pleasecreate a support thread or PM me on the MyBB support forums.


So I think that's all you'll need. If you still got any concerns or problems don't hesitate to ask. Thanks for downloading and reading.
